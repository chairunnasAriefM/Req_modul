## Getting Started

PHP Ver : PHP 8.1.2

1. Install the dependencies

```shell
composer install
```

2. Copy `env to `.env`

```shell
cp env .env
```

or kalau di windows

```shell
copy env .env
```

3. Start the webserver

```shell
php spark serve
```
